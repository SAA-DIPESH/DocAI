# Sentence Rephraser Agent Specification

Input: A paragraph from a tender document.

Output: A professionally rephrased paragraph.

Requirements:

* Improve readability, sentence flow, and clarity.
* Preserve technical vocabulary and domain-specific terms.
* Maintain the original intent and factual content.
* Keep formatting, values, references, and units unchanged.
* Do not summarise, expand, or introduce new information.
* Produce deterministic, proposal-quality output suitable for downstream processing and embedding.
